# Math Didactics Hub

Diese App ist für GitHub Pages vorkonfiguriert.

## Deployment über den GitHub-Link

1. Repository nach GitHub pushen.
2. In GitHub unter **Settings → Pages** als Quelle **GitHub Actions** auswählen.
3. Auf den `main`-Branch pushen (oder den Workflow manuell unter **Actions** starten).
4. Nach erfolgreichem Workflow ist die Seite verfügbar unter:
   - `https://<github-user>.github.io/<repo-name>/`

## Lokal starten

```bash
npm install
npm run dev
```

## Lokal Build testen

```bash
npm run build
npm run preview
```
